# Dossier Public - annonceauto.ci

Ce dossier contient les fichiers statiques accessibles publiquement.

## Structure recommandée :

```
public/
├── favicon.ico          # Icône du site
├── logo.png             # Logo du site
├── robots.txt           # Configuration pour les moteurs de recherche
├── sitemap.xml          # Plan du site (SEO)
└── images/              # Images statiques
    ├── placeholder.png  # Image par défaut
    └── ...
```

## Utilisation

Les fichiers dans ce dossier sont accessibles depuis la racine du site :
- `/favicon.ico` → `public/favicon.ico`
- `/logo.png` → `public/logo.png`
- `/images/placeholder.png` → `public/images/placeholder.png`

## Fichiers à ajouter (recommandé)

Pour compléter votre site, ajoutez :
- **logo.png** - Le logo de votre site (format PNG, 200x200px minimum)
- **og-image.png** - Image pour les partages sur réseaux sociaux (1200x630px)
- **robots.txt** - Configuration SEO
- **sitemap.xml** - Plan du site pour Google

Ces fichiers amélioreront le référencement et l'apparence de votre site.
