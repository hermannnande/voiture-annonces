### 💡 ASTUCE : Transférer sans node_modules ###

Pour gagner du temps, vous pouvez :
1. Ne PAS transférer les dossiers node_modules/
2. Une fois sur le serveur, exécuter :
   ```bash
   cd /home/annonceauto/backend && npm install
   cd /home/annonceauto/frontend && npm install
   ```

Cela sera plus rapide que de transférer des milliers de petits fichiers !
